# DEVersidade IA
O objetivo do projeto foi criar uma assistente de IA Web simples, desenvolvida do zero com HTML, CSS e JavaScript puro.
Permitindo  que um usuário interaja com a API do Google Gemini para fazer perguntas e receber respostas diretamente no navegador.

## Novas Funcionalidades:

- Botões para COPIAR e LIMPAR resposta;
- Melhorias na interface e
- Melhorias na responsividade.

## Tecnologias utilizadas:
- Html;
- CSS;
- JavaScript e
- Google Gemini API.

## Como funciona:
Ao acessar a página do projeto no GitHub Pages: https://mavcostta.github.io/Projeto6/

Insira a chave* API no campo apropriado e digite sua pergunta.

Clique em "PERGUNTAR" e veja a resposta fornecida pela IA.

*(Obtenha uma chave de API do Google Gemini)

# 🤖 IA DEVersidade

Um assistente de Inteligência Artificial simples que interage com a **API Gemini** da Google, permitindo ao usuário fazer perguntas e receber respostas diretamente no navegador.

# Link da página para teste

- https://mavcostta.github.io/Projeto6/

## 📌 Funcionalidades

- Inserir **Chave de API** diretamente na interface.
- Enviar perguntas para o modelo **Gemini 2.0 Flash**.
- Exibir respostas processadas em tempo real.
- Interface simples e minimalista, feita em HTML, CSS e JavaScript puro.

## 🛠 Tecnologias Utilizadas

- **HTML5** para estrutura da página.
- **CSS3** para estilização.
- **JavaScript** para interação e requisição à API.
- **Google Gemini API** para processamento de linguagem natural.
