# 🤖 IA DEVersidade

Um assistente de Inteligência Artificial simples que interage com a **API Gemini** da Google, permitindo ao usuário fazer perguntas e receber respostas diretamente no navegador.

## 📌 Funcionalidades

- Inserir **Chave de API** diretamente na interface.
- Enviar perguntas para o modelo **Gemini 2.0 Flash**.
- Exibir respostas processadas em tempo real.
- Interface simples e minimalista, feita em HTML, CSS e JavaScript puro.

## 🛠 Tecnologias Utilizadas

- **HTML5** para estrutura da página.
- **CSS3** para estilização.
- **JavaScript** para interação e requisição à API.
- **Google Gemini API** para processamento de linguagem natural.
